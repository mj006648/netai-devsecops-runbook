# Templates

| Template | Use case |
| --- | --- |
| [Runbook template](runbook-template.md) | 일반 운영 절차 또는 복구 기록 |
| [Incident template](incident-template.md) | 장애 기록, postmortem, 재발 방지 |
| [Change plan template](change-plan-template.md) | 계획된 점검, 업그레이드, migration |
| [Hardware troubleshooting template](hardware-troubleshooting-template.md) | PSU, rack, NIC, BMC/IPMI, cabling, vendor support 이슈 |
