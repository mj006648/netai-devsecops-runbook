# Karmada ApplicationSet manifests

이 디렉터리는 ArgoCD ApplicationSet으로 여러 Karmada Application을 생성하는 실험 manifest를 보관한다.

## karmada-demo-appset

```text
ApplicationSet: karmada-demo-appset
Generated Applications:
- karmada-appset-edge -> kubernetes/karmada/manifests/demo-appset-edge
- karmada-appset-data -> kubernetes/karmada/manifests/demo-appset-data
Destination:
- Karmada API Server
```
